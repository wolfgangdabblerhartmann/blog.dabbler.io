倒计时一百天（DAY100）

发表于 2020-02-22

“人生苦短 我用 Python”这句话来自于 ANSI C++协会成员之一的 Bruce Eckel，原话为“Life is short, you need Python”。相信很多 C++、Java 开发者都看过这位大牛的书，他的话是绝对具备权威的。他只是简单的说出了心里话，从此这句话被传遍大江南北。

上面这一段，摘录自全栈弄潮儿的文章《Python - 100 天从新手到大师》。Bruce Eckel 这位大牛是谁我不认识、他的书我没看过、他权不权威我不知道，这句话挺火我经常看到。

上面这个就是 Python 语言的 LOGO。
